Built a production-ready ML pipeline using census data, covering data cleaning, model training, and performance analysis across data slices. Exposed the model via a FastAPI REST API with GET and POST endpoints for inference, with automated testing (pytest) and code quality checks (flake8) running on every push through GitHub Actions.
